Uses Alpha Beta Algorithm*

An optmisation of the MiniMax Algorithm, the Alpha Beta Algorithm makes the assumption that each side will make only the best move available to them. This allows it to dismiss suboptimal moves as soon as superior alternatives are found. Unfortunately, only static evaluation is currently implemented.
